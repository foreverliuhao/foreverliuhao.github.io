/**

  • 爬楼梯问题
  • @author Liuhao
    题目描述：假设有n阶台阶，每次只能上一节台阶或者两者台阶，问上n阶台阶一共有都多少种 方法
    解题思路： /
    public class ClimbStairs {
    public static int climbStairs(int n){
    int[] dp=new int[n];
dp[0]=1;
dp[1]=2;
for(int i=2;i<n;i++) {
    dp[i]=dp[i-1]+dp[i-2];
}
return dp[n-1];
    }
    public static void main(String[] args) {
    Scanner scan = new Scanner(System.in);
System.out.println("请输入楼梯的阶数");
int n=scan.nextInt();
int method=climbStairs(n);
System.out.println(method);
    }
    }
