2018-3-31链表倒数第K个节点

Posted on 2018-04-02

基本思想：链表的长度为k，要删除倒数第k个节点，由此我们可以得到
倒数第k个节点的前驱节点为N-k,那么从k开始遍历链表，K的值就会变为
K-N，第二次遍历时，K的值不断加一，加到零就停止遍历。
public class Node{
public int value;
public Node next;
public Node(int data){
this.value=data;
}

}
public Node removeKthNode(Node head,int lastKth){
if(head==null||lastKth<1){
return head;
}
Node cur=head;
while(cur!=null){
lastKth–;
cur=cur.next;
}
if(lastKth==0){
head=head.next;
}
if(lastKth<0){
cur=head;
while(++lastKth!=0){
cur=cur.next;
}
cur.next=cur.next.next;

}
return head;
}
